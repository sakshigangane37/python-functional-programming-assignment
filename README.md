# Python Functional Programming Assignment

This repository contains Python programs created to practice lambda functions and functional programming concepts using `filter()`, `map()`, and `reduce()`.

## Programs Included

1. Calculate the power of two of a number using a lambda function.
2. Multiply two numbers using a lambda function.
3. Filter numbers between 70 and 90, increase each number by 10, and calculate the product using `reduce()`.
4. Filter even numbers, calculate their squares using `map()`, and return their addition using `reduce()`.
5. Filter prime numbers, multiply each prime number by 2, and return the maximum number using `reduce()`.

## Concepts Practiced

- Lambda Functions
- filter()
- map()
- reduce()
- Python Lists
- User Input
- List Traversal
- Conditional Expressions
- Prime Number Checking
- Functional Programming
